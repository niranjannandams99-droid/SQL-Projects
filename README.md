## 🍜 Project 01 — Danny's Diner Case Study

<img src="https://img.shields.io/badge/Domain-Restaurant_Analytics-red?style=flat-square"/> <img src="https://img.shields.io/badge/Tool-MySQL-4479A1?style=flat-square&logo=mysql"/>

Danny opened a small Japanese restaurant and wants to use customer data to understand visiting patterns, spending habits, and favourite menu items to decide whether to expand his loyalty program.

**10 Business Questions Solved:**
- Total spending per customer
- Visit frequency analysis
- First & last item purchased
- Most popular menu items
- Loyalty points calculation with multipliers

**Key Concepts:** `CTEs` · `ROW_NUMBER()` · `RANK()` · `DENSE_RANK()` · `JOINS` · `CASE WHEN` · `DATE_ADD()`

## 🛒 Project 02 — E-Commerce Customer Intelligence System

<img src="https://img.shields.io/badge/Domain-Retail_Analytics-purple?style=flat-square"/> <img src="https://img.shields.io/badge/Tool-MySQL-4479A1?style=flat-square&logo=mysql"/> <img src="https://img.shields.io/badge/Dataset-500K+_Records-orange?style=flat-square"/>

A complete end-to-end business intelligence system built on 500K+ real-world e-commerce transactions from Kaggle. Simulates a production-grade retail analytics environment.

**10-Module Analytical Pipeline:**

| Module | Description |
|--------|-------------|
| 01 Database Setup | Schema design & indexing |
| 02 Data Cleaning | Null handling, type conversion, validation |
| 03 Data Transformation | Revenue columns, feature engineering |
| 04 Revenue Analysis | Monthly trends, top customers, top products |
| 05 RFM Segmentation | Champions, Loyal, Big Spenders, At-Risk |
| 06 Cohort Analysis | Monthly retention rates & trends |
| 07 Business KPIs | Executive-level reporting metrics |
| 08 SQL Views | Reusable reporting views |
| 09 Stored Procedures | Automated business reporting |
| 10 Final Insights | Strategic recommendations |

**Key Concepts:** `RFM Modeling` · `Cohort Retention` · `Window Functions` · `CTEs` · `Views` · `Stored Procedures` · `KPI Reporting` · `Data Cleaning`
